About the Project
Concrete Compressive Strength Prediction

The project has been developed to predict the compressive strength of concrete based on the quantity of it's mixture properties such as cement, water, fly ash, coarse aggregate, fine aggregate, superplasticizer, blast furnace and also the age of the concrete.. The dataset can be found here. A Flask webapp have been developed for this project. The web app is deployed in heroku. Libraries such as pandas, numpy, seaborn, matplotlib, dvc and pycaret have been used to develop this project. Project Documentation can be found here.
